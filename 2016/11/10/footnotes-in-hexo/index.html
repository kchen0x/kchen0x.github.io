Markdown 的脚注

Markdown 基本语法中并不包含脚注[1]语法，但是脚注作为一种常见的文本格式，对于文字编辑工作者，特别是像我这种喜欢插入引文的人而言，有着很大的使用需求。所以 Multi-Markdown[2] 在其扩充语法集中增添了脚注的语法：

插件的安装和使用

MMD 的脚注语法得到广泛的传播和认可，大部分的 Markdown 编辑器现在都采用了该语法来渲染脚注。可是 Hexo 的默认渲染器是不支持脚注语法的，所以我写了这个简单的 Reference 插件来实现脚注的渲染。该插件已收录于 Hexo 官方插件页。插件的安装和使用非常的简单，只需要进入博客目录，然后安装：
