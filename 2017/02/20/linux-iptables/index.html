Linux - Iptable 的使用

2017-02-20 阅读次数 0

ipatable 是 Linux 内核自带的防火墙解决方案。它通过对经由网络的每一个包进行一系列的规则验证来决定该如何处理。

我们可以匹配协议类型、源地址和目的地址或者端口、接口以及它们和之前的包的关系等等。这些规则会被进一步的结构化成为规则链，网络包将会顺序地被规则链中的规则一一检查。它们可以被按需生成，但是有3种默认的规则：

  • INPUT：处理流入服务器的流量。
  • OUTPUT：处理流出服务的流量。
  • FORWARD：处理流经服务器的流量而不停止它们。
